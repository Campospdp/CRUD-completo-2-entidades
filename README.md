Projeto de Gerenciamento de Alunos - SENAI
Este repositório contém a implementação das funcionalidades de CRUD (Create, Read, Update, Delete) para a entidade Aluno, desenvolvida durante as aulas do curso de programação. O foco principal desta etapa é a aplicação de regras de negócio estritas e validações de dados para garantir a integridade das informações no sistema.
